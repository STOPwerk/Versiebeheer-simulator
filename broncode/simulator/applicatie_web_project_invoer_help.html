Deze pagina simuleert software die bevoegd gezag en adviesbureaus gebruiken om regelgeving te maken en besluiten op te stellen. De simulatie laat zien hoe dat zou kunnen werken, niet hoe de software daadwerkelijk functioneert - het staat de software leveranciers en hun klanten vrij om het helemaal anders in te richten. Deze simulator laat zien dat het mogelijk is om de technische vereisten van het versiebeheer zoals dat via STOP uitwisseld moet worden, voor de eindgebruikers te verbergen. Het geeft een indruk welke informatie de software moet bijhouden om aan de STOP-vereisten te kunnen voldoen.

In de simulator wordt een scenario opgebouwd, bestaande uit een serie activiteiten die de "eindgebruikers" met behulp van de "software" uitvoeren. De simulator laat de interactie zien tussen eindgebruikers en de software. Het scenario kan opgestuurd worden naar de LVBB simulator.
